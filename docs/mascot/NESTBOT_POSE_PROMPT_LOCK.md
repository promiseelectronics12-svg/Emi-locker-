# NestBot Pose Prompt Lock

Use this prompt structure whenever creating new NestBot poses or animation frames.

## Required Prompt

Create the exact same NestBot mascot from the approved EMI Locker reference.

Preserve all identity details:

- Rounded square robot head.
- Dark green glass face panel.
- Large glowing mint eyes.
- Friendly mouth style.
- Metallic green armor.
- Gold and green shield-lock chest emblem.
- Side helmet/ear pods.
- Small antenna/signal element above the head.
- Compact body proportions.
- Sturdy green boots.
- Premium polished 3D rendering style.

Change only:

- Pose.
- Hand gesture.
- Facial expression within the same face style.
- Held prop.

Do not redesign the mascot. Do not change the color palette, head shape, face shape, armor pattern, chest emblem, proportions, or rendering style.

## Negative Prompt

new robot design, different mascot, different head, different face, different eyes, different color palette, different armor, different chest logo, different body proportions, flat cartoon style, anime style, aggressive style, square frame, fake checkerboard background, cropped body, extra limbs, unreadable prop, text artifacts

## Pose Examples

- NestBot waving hello.
- NestBot holding a six-digit activation code.
- NestBot holding a shield with a check mark.
- NestBot pointing to a map pin.
- NestBot showing a payment reminder.
- NestBot holding a lock/unlock status badge.
- NestBot warning about a missing permission.

## Review Rule

If the generated pose looks like a new character, reject it immediately. A pose is only accepted when it still feels like the same NestBot at first glance.
